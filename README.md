# BlueReyd HealthTech Solutions

**Clinician-Led Digital Innovation for Real-World Healthcare**

BlueReyd is a clinician-product architecture studio building reliable, offline-first mHealth solutions for Nigeria and the developing world. This repository contains the official BlueReyd landing page — a production-grade Next.js web application.

---

## Live Site

🌐 [bluereyd.vercel.app](https://bluereyd.vercel.app)

---

## About

BlueReyd bridges the gap between high-stakes clinical expertise and full-stack technical excellence. Every product we ship is:

- **Clinician-designed** — built from direct patient care experience
- **Offline-first** — reliable at the bedside with zero network dependency
- **Localized** — designed for the realities of Nigerian and African healthcare

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Vanilla CSS with CSS custom properties
- **Fonts:** Inter (Google Fonts via next/font)
- **Deployment:** Vercel

---

## Features

- ⚡ Dark / Light mode with system preference detection and `localStorage` persistence
- 📱 Fully responsive — mobile-first layout
- 🎨 Glassmorphism UI with premium micro-animations
- ♿ Accessible — semantic HTML, ARIA labels, keyboard navigation
- 🔒 No trackers, no third-party analytics

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.
